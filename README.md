### MISSFRESH
### 项目运行
```
1. 打开命令行窗口，运行以下命令开启服务
git clone https://github.com/792884274/MISSFRESH.git

cd node-MISSFRESH

cnpm install

set NODE_ENV=development

node index.js
2. 打开另一个命令行窗口，运行以下命令开启vue
cd vue-MISSFRESH

cnpm install

npm run dev
```

### 注意
如果运行是出现接口占用问题，请将两个文件夹中的端口‘3396’都改成其他的端口，例如‘3397’
